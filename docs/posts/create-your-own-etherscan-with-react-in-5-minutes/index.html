Create your own Etherscan with React in 5 minutes

In this tutorial, we'll create a simple single-page etherscan in reactjs. For the clean UI, we are going to use the amazing semantic-ui-react which is the official react integration for semantic-ui.

You can find the complete code in the github.

Without wasting any time let's build it.

Prerequisite

  • nodejs

  • Code editor – (I am using VS Code)

I am assuming that you have a basic understanding of reactjs and components.

Create a react project

We are using create-react-app as a boilerplate. Follow this link to learn more about it.

Open a terminal and run

1// this command will install create-react-app
2npx create-react-app etherscan-react

If you've previously installed create-react-app globally via npm install -g create-react-app, we recommend you uninstall the package using npm uninstall -g create-react-app to ensure that npx always uses the latest version.

etherscan-react is the name of the project. You can name it anything you like.

Once it is installed, open the terminal in the project and run npm start.

Go to localhost:3000. You can see the boilerplate homepage.

Now, our react boilerplate is ready.

Install Dependencies

Following packages are required in this project.

semantic-ui-react:

Semantic UI React is the official React integration for Semantic UI.

1npm install semantic-ui-react

axios:

To make http request we are using axios.

1npm install axios

Etherscan API

We are going to use Etherscan API to get the ethereum blockchain data like the latest block details, transaction detail, ether price etc.

Follow this link to register for an Etherscan account. Sign in if you already registered.

Go to the API-Keys section on the left side.

Create a new api key token. Give eth-react as the app name. You can give it any name.

Note down the API-KEY.

Let's start the fun part, the coding

Open the project in your favourite code editor. I am using VS Code.

Create a components folder inside the src. Inside the components create the following components.

- src
	- components
		- Eth-Overview
		- Header
		- Latest-Blocks
		- Latest-Txs

  • Eth-Overview: The overview of the ethereum blockchain such as ether price, latest block no., difficulty, and market cap.

  • Header: Simple header of the application.

  • Latest-Blocks: This component will fetch the latest 5 blocks.

  • Latest-Txs: This component will fetch all the transactions in the latest block.

  1import React, { Component } from "react";
  2import axios from "axios";
  3import "./eth-overview.css";
  4import { Card, Grid, Icon } from "semantic-ui-react";
  5import LatestBlocks from "../Latest-Blocks/index";
  6import LatestTxs from "../Latest-Txs/index";
  7
  8// import api key from the env variable
  9const apiKey = process.env.REACT_APP_ETHERSCAN_API_KEY;
 10
 11const endpoint = `https://api.etherscan.io/api`;
 12
 13class EthOverview extends Component {
 14  constructor() {
 15    super();
 16    this.state = {
 17      ethUSD: "",
 18      ethBTC: "",
 19      blockNo: "",
 20      latestBlock: 0,
 21      difficulty: "",
 22      marketCap: 0
 23    };
 24  }
 25
 26  async componentDidMount() {
 27    // get the ethereum price
 28    axios
 29      .get(endpoint + `?module=stats&action=ethprice&apikey=${apiKey}`)
 30      .then(res => {
 31        const { result } = res.data;
 32        this.setState(
 33          {
 34            ethUSD: result.ethusd,
 35            ethBTC: result.ethbtc
 36          },
 37          () => {
 38            // get the market cap of ether in USD
 39            axios
 40              .get(endpoint + `?module=stats&action=ethsupply&apikey=${apiKey}`)
 41              .then(res => {
 42                const { result } = res.data;
 43                // in wei
 44                const priceWei = result.toString();
 45
 46                // in ether
 47                const priceEth = priceWei.slice(0, priceWei.length - 18);
 48                console.log(result, priceWei, priceEth);
 49                // convert eth in USD
 50                this.setState({
 51                  marketCap: parseInt(priceEth) * this.state.ethUSD
 52                });
 53              });
 54          }
 55        );
 56      });
 57
 58    // get the latest block number
 59    axios
 60      .get(endpoint + `?module=proxy&action=eth_blockNumber&apikey=${apiKey}`)
 61      .then(res => {
 62        this.setState({
 63          latestBlock: parseInt(res.data.result),
 64          blockNo: res.data.result // save block no in hex
 65        });
 66
 67        // get the block difficulty
 68        axios
 69          .get(
 70            endpoint +
 71              `?module=proxy&action=eth_getBlockByNumber&tag=${res.data.result}&boolean=true&apikey=${apiKey}`
 72          )
 73          .then(blockDetail => {
 74            const { result } = blockDetail.data;
 75
 76            const difficulty = parseInt(result.difficulty).toString();
 77
 78            // convert difficulty in Terra Hash
 79            // instead of dividing it with 10^12 we'll slice it
 80            const difficultyTH = `${difficulty.slice(0, 4)}.${difficulty.slice(
 81              4,
 82              6
 83            )} TH`;
 84
 85            this.setState({
 86              difficulty: difficultyTH
 87            });
 88          });
 89      });
 90  }
 91
 92  getLatestBlocks = () => {
 93    if (this.state.latestBlock) {
 94      return <LatestBlocks latestBlock={this.state.latestBlock}></LatestBlocks>;
 95    }
 96  };
 97
 98  getLatestTxs = () => {
 99    if (this.state.blockNo) {
100      return <LatestTxs blockNo={this.state.blockNo}></LatestTxs>;
101    }
102  };
103
104  render() {
105    const { ethUSD, ethBTC, latestBlock, difficulty, marketCap } = this.state;
106    return (
107      <div>
108        <Grid>
109          <Grid.Row>
110            <Grid.Column width={4}>
111              <Card>
112                <Card.Content>
113                  <Card.Header style={{ color: "#1d6fa5" }}>
114                    <Icon name="ethereum"></Icon> ETHER PRICE
115                  </Card.Header>
116                  <Card.Description textAlign="left">
117                    <Icon name="usd"></Icon>
118                    {ethUSD} <Icon name="at"></Icon> {ethBTC}{" "}
119                    <Icon name="bitcoin"></Icon>
120                  </Card.Description>
121                </Card.Content>
122              </Card>
123            </Grid.Column>
124            <Grid.Column width={4}>
125              <Card>
126                <Card.Content>
127                  <Card.Header style={{ color: "#1d6fa5" }}>
128                    <Icon name="list alternate outline"></Icon> LATEST BLOCK
129                  </Card.Header>
130                  <Card.Description textAlign="left">
131                    <Icon name="square"></Icon> {latestBlock}
132                  </Card.Description>
133                </Card.Content>
134              </Card>
135            </Grid.Column>
136            <Grid.Column width={4}>
137              <Card>
138                <Card.Content>
139                  <Card.Header style={{ color: "#1d6fa5" }}>
140                    <Icon name="setting"></Icon> DIFFICULTY
141                  </Card.Header>
142                  <Card.Description textAlign="left">
143                    {difficulty}
144                  </Card.Description>
145                </Card.Content>
146              </Card>
147            </Grid.Column>
148            <Grid.Column width={4}>
149              <Card>
150                <Card.Content>
151                  <Card.Header style={{ color: "#1d6fa5" }}>
152                    <Icon name="world"></Icon> MARKET CAP
153                  </Card.Header>
154                  <Card.Description textAlign="left">
155                    <Icon name="usd"></Icon> {marketCap}
156                  </Card.Description>
157                </Card.Content>
158              </Card>
159            </Grid.Column>
160          </Grid.Row>
161        </Grid>
162
163        <Grid divided="vertically">
164          <Grid.Row columns={2}>
165            <Grid.Column>{this.getLatestBlocks()}</Grid.Column>
166            <Grid.Column>{this.getLatestTxs()}</Grid.Column>
167          </Grid.Row>
168        </Grid>
169      </div>
170    );
171  }
172}

Don't panic, let me breakdown the code for you so that you can understand.

  • Line 4: Importing Grid component for the dashboard layout. There are 2 rows, the first row will show the etherprice in USD and Bitcoin. The second row is for the latest blocks and the latest transactions. Card component is used to show the ether price, difficulty, latest block and market cap. Icon component is for the icons we require in the dashboard, such as $.

  • Line 5: LatestBlocks component will return the table of latest 5 blocks using the etherscan api. latestBlock number will send as param to this component.

  • Line 6: LatestTxs component will return the last 5 transactions of the latest block. blockNo (latestBlock) in hex format will be sent as a param.

  • Line 9: For testing purpose you can paste your api key and check the functionality. But it is recommended to get the api key from the environment variable.

  • Line 11: endpoint for all the requests.

  • Line 26: ComponentDidMount is a special function in the react, which will run first for that particular component. You can think of it as a constructor of the component. The comments explained everything in the code. It is fetching and calculating the basic information like ether price, latest block, difficulty and market cap.

  • Line 92: The getLatestBlocks method will render the LatestBlocks component.

  • Line 98: The getLatestTxs method will render the LatestTxs component.

Latest-Blocks.jsx

Open the Latest-Blocks.jsx in the code editor and paste the below code.

 1import React, { Component } from "react";
 2import {
 3  Table,
 4  Label
 5} from "semantic-ui-react";
 6
 7import axios from "axios";
 8
 9const apiKey = process.env.REACT_APP_ETHERSCAN_API_KEY;
10const endpoint = `https://api.etherscan.io/api`;
11
12class LatestBlocks extends Component {
13  constructor(props) {
14    super(props);
15    this.state = {
16      blocks: []
17    };
18  }
19
20  componentDidMount = () => {
21    this.getBlocks();
22  };
23
24  getBlocks = async () => {
25    const { latestBlock } = this.props;
26
27    let blocks = [];
28
29    for (let i = 0; i < 5; i = i + 1) {
30      // get the block transaction
31      const blockDetail = await axios.get(
32        endpoint +
33          `?module=proxy&action=eth_getBlockByNumber&tag=${(
34            latestBlock - i
35          ).toString(16)}&boolean=true&apikey=${apiKey}`
36      );
37
38      const { result } = blockDetail.data;
39      blocks.push(
40        <Table.Row key={i}>
41          <Table.Cell>
42            <Label color="blue">Bk</Label> {latestBlock - i}
43          </Table.Cell>
44          <Table.Cell>
45            Miner {result.miner} <br></br>
46            Txs {result.transactions.length}
47          </Table.Cell>
48          <Table.Cell>
49            <Label color="blue">Size </Label> {parseInt(result.size)} bytes
50          </Table.Cell>
51        </Table.Row>
52      );
53
54      this.setState({
55        blocks: blocks
56      });
57    }
58  };
59
60  render() {
61    return (
62      <Table fixed>
63        <Table.Header>
64          <Table.Row>
65            <Table.Cell style={{ color: "#1d6fa5" }}>
66              <h4>Latest Blocks</h4>
67            </Table.Cell>
68          </Table.Row>
69        </Table.Header>
70
71        <Table.Body>{this.state.blocks}</Table.Body>
72      </Table>
73    );
74  }
75}
76
77export default LatestBlocks;

This component will fetch the latest 5 blocks from the etherscan and returns the table. You can increase the no. of blocks to fetch inline 29.

 1import React, { Component } from "react";
 2import {
 3  Table,
 4  Label
 5} from "semantic-ui-react";
 6import axios from "axios";
 7
 8const apiKey = process.env.REACT_APP_ETHERSCAN_API_KEY;
 9const endpoint = `https://api.etherscan.io/api`;
10
11class LatestTxs extends Component {
12  constructor(props) {
13    super(props);
14    this.state = {
15      transactions: []
16    };
17  }
18
19  componentDidMount = () => {
20    this.getTxs();
21  };
22
23  getTxs = async () => {
24    const { blockNo } = this.props;
25
26    // get the block transaction
27    const blockDetail = await axios.get(
28      endpoint +
29        `?module=proxy&action=eth_getBlockByNumber&tag=${blockNo}&boolean=true&apikey=${apiKey}`
30    );
31
32    const { transactions } = blockDetail.data.result;
33
34    let txsDetails = [];
35
36    for (let i = 0; i < 5; i = i + 1) {
37      const tx = transactions[i];
38      txsDetails.push(
39        <Table.Row key={i}>
40          <Table.Cell>
41            <Label color="blue">Tx</Label> {tx.hash}
42          </Table.Cell>
43          <Table.Cell>
44            From {tx.from} <br></br>
45            To {tx.to}
46          </Table.Cell>
47          <Table.Cell>
48            {" "}
49            <Label color="blue">Eth</Label> {parseInt(tx.value) / 10 ** 18}
50          </Table.Cell>
51        </Table.Row>
52      );
53    }
54
55    this.setState({
56      transactions: txsDetails
57    });
58  };
59
60  render() {
61    return (
62      <div>
63        <Table fixed>
64          <Table.Header>
65            <Table.Row>
66              <Table.Cell style={{ color: "#1d6fa5" }}>
67                <h4> Latest Transactions</h4>
68              </Table.Cell>
69            </Table.Row>
70          </Table.Header>
71
72          <Table.Body>{this.state.transactions}</Table.Body>
73        </Table>
74      </div>
75    );
76  }
77}
78
79export default LatestTxs;

This component will fetch the latest 5 transactions of the latest block from the etherscan and returns the table. You can increase the no. of transactions to fetch inline 36.
