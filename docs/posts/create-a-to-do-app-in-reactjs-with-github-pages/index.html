Create a To Do App in Reactjs with Github Pages

Tuesday, April 14, 2020 • 10 minutes to read

TutorialReactjs2020tutorialreactjsgithub-pages

In this tutorial, we will create a to-do list app in reactjs. We will use browser local storage to save the tasks, which will keep the tasks even after we close the browser. In the end, we will host this application on Github pages.

Pre-requisites

  1. Nodejs
  2. Code editor (I am using VS Code)
  3. Github Account

React Application

We are going to use create-react-app as our base application.

1npx create-react-app my-task-list
2cd my-task-list

If you've previously installed create-react-app globally via npm install -g create-react-app, we recommend you uninstall the package using npm uninstall -g create-react-app to ensure that npx always uses the latest version.

Install semantic-ui-react

For this project, we are using semantic-ui css. Using the only css is a bit headache where you have to configure each element in the web page. To make this quick and less hassle, we have semantic-ui-react library. Semantic UI React provides the react component which has already configured semantic-ui css. In short, you don't have to align a component or adjust the margin.

Semantic UI React is the official React integration for Semantic UI.

Open the terminal inside the my-task-list project and run the below command.

1npm install semantic-ui-react

My-Task-List.jsx

In this file, we are going to define our my-task-list component. For storage, we are using the browser's local storage.

Browser Local Storage

This local storage stores a key-value pair. The key and value both are strings. While saving we have to convert the object into a string and while working on it we have to convert the string into an object.

To access the local storage we will use the localStorage object. localStorage has given to 2 methods to set and get the item.

  • setItem: To save the key-value in the local storage.
1localStorage.setItem(key, value);
  • getItem: Get the value by its key from the local storage.
1localStorage.getItem(key)

Open the My-Task-List.jsx paste the below code.

  1import React, { Component } from "react";
  2import { Card, Header, Form, Input, Icon } from "semantic-ui-react";
  3import "./my-task-list.css";
  4
  5class MyTaskList extends Component {
  6  constructor(props) {
  7    super(props);
  8
  9    this.state = {
 10      task: "",
 11      tasklist: []
 12    };
 13  }
 14
 15  // on load get the task list
 16  componentDidMount = () => {
 17    this.getTasks();
 18  };
 19
 20  onChange = event => {
 21    this.setState({
 22      [event.target.name]: event.target.value
 23    });
 24  };
 25
 26  // add task to the list
 27  onSubmit = () => {
 28    // check is task is empty string
 29    if (this.state.task) {
 30      // get the task list from the local storage
 31      let tasklist = JSON.parse(localStorage.getItem("tasklist"));
 32
 33      // task list is null means empty
 34      // create an empty list
 35      if (tasklist == null) {
 36        tasklist = [];
 37      }
 38
 39      // create task object
 40      // default status is false
 41      let task = {
 42        task: `📄 ${this.state.task}`,
 43        status: false
 44      };
 45
 46      // add the task to the task list
 47      tasklist.push(task);
 48
 49      // save the task list in the local storage
 50      localStorage.setItem("tasklist", JSON.stringify(tasklist));
 51
 52      // clear the form
 53      this.setState({ task: "" });
 54
 55      // refresh the tasks
 56      this.getTasks();
 57    }
 58  };
 59
 60  // get all the tasks
 61  getTasks = () => {
 62    // get the task list from the local storage
 63    let tasklist = JSON.parse(localStorage.getItem("tasklist"));
 64
 65    // check if task list is empty
 66    if (tasklist) {
 67      // sort all the tasks on the basis of status
 68      // completed task will move down
 69      tasklist = tasklist.sort((a, b) => {
 70        if (a.status) {
 71          return 1;
 72        } else if (b.status) {
 73          return -1;
 74        }
 75        return 0;
 76      });
 77
 78      // save the task list in the local storage
 79      localStorage.setItem("tasklist", JSON.stringify(tasklist));
 80
 81      // set the tasklist to the state
 82      this.setState({
 83        // default color
 84        // Incomplete: yellow
 85        // complete: green
 86        tasklist: tasklist.map((item, index) => {
 87          let color = "yellow";
 88          let cardBackground = { background: "white" };
 89          let taskComplete = { textDecoration: "none" };
 90
 91          if (item.status) {
 92            color = "green";
 93            cardBackground.background = "beige";
 94            taskComplete["textDecoration"] = "line-through";
 95          }
 96          return (
 97            <Card key={index} color={color} fluid style={cardBackground}>
 98              <Card.Content>
 99                <Card.Header textAlign="left" style={taskComplete}>
100                  <div style={{ wordWrap: "break-word" }}>{item.task}</div>
101                </Card.Header>
102
103                <Card.Meta textAlign="right">
104                  <Icon
105                    link
106                    name="check circle"
107                    color="green"
108                    onClick={() => this.updateTask(index)}
109                  />
110                  <span style={{ paddingRight: 10 }}>Done</span>
111                  <Icon
112                    link
113                    name="undo"
114                    color="yellow"
115                    onClick={() => this.undoTask(index)}
116                  />
117                  <span style={{ paddingRight: 10 }}>Undo</span>
118                  <Icon
119                    link
120                    name="delete"
121                    color="red"
122                    onClick={() => this.deleteTask(index)}
123                  />
124                  <span style={{ paddingRight: 10 }}>Delete</span>
125                </Card.Meta>
126              </Card.Content>
127            </Card>
128          );
129        })
130      });
131    }
132  };
133
134  // update the task status to true
135  updateTask = index => {
136    // get the task list from the local storage
137    let tasklist = JSON.parse(localStorage.getItem("tasklist"));
138    // change status to true
139    tasklist[index].status = true;
140    // save the updated task list
141    localStorage.setItem("tasklist", JSON.stringify(tasklist));
142    // refresh the task list
143    this.getTasks();
144  };
145
146  // undone the task status from true to false
147  undoTask = index => {
148    // get the task list from the local storage
149    let tasklist = JSON.parse(localStorage.getItem("tasklist"));
150    // change status to false
151    tasklist[index].status = false;
152    // save the updated task list
153    localStorage.setItem("tasklist", JSON.stringify(tasklist));
154    // refresh the task list
155    this.getTasks();
156  };
157
158  // delete the task from the task list
159  deleteTask = index => {
160    // get the task list from the local storage
161    let tasklist = JSON.parse(localStorage.getItem("tasklist"));
162    // remove the task from the task list
163    tasklist.splice(index, 1);
164    // save the updated task list
165    localStorage.setItem("tasklist", JSON.stringify(tasklist));
166    // refresh the task list
167    this.getTasks();
168  };
169
170  render() {
171    return (
172      <div>
173        <div>
174          <Header as="h1">
175            <div className="app-header">📝 My Task List</div>{" "}
176          </Header>
177        </div>
178        <div className="app-form">
179          <Form onSubmit={this.onSubmit}>
180            <Input
181              type="text"
182              name="task"
183              onChange={this.onChange}
184              value={this.state.task}
185              fluid
186              placeholder="task..."
187            />
188          </Form>
189        </div>
190        <div>
191          <Card.Group>{this.state.tasklist}</Card.Group>
192        </div>
193      </div>
194    );
195  }
196}
197
198export default MyTaskList;
199

The key to the tasks is tasklist.

A quick explanation of the code.

  • For view we are using Card, Header, Form, Input, Icon react components from the semantic-ui-react. Go to the links to learn more.
  • getTasks: This function will get the tasks from the local storage and create the task list using Card component. Each task has 3 actions, Done, Undo, and Delete.
  • onSubmit: This function will save the task in the local storage.
  • updateTask: This function will turn the task status to true in the local storage. It means the task is complete. This function will trigger on click of Done.
  • undoTask: This function will turn the task status to false in the local storage. It means the task is incomplete. This function will trigger on click of Undo.
  • deleteTask: This function will delete/remove the task from the local storage. This function will trigger on click of Delete.
