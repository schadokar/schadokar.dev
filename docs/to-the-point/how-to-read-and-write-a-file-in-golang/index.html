How to Read and Write a File in Golang?

Tuesday, March 24, 2020 â€¢ 9 minutes to read

Tutorialto-the-pointGolanggolangto-the-point

In this tutorial, we will learn how to read and write files in the file system using the io/ioutil package.

Write a file using WriteFile

The WriteFile function takes 3 arguments, filename, data in a byte array, and filemode.

1func WriteFile(filename string, data []byte, perm os.FileMode) error

A FileMode represents a file's mode and permission bits. Learn more

 1package main
 2
 3import (
 4    "io/ioutil"
 5    "log"
 6)
 7
 8func main() {
 9    // create a byte array of a string
10    data := []byte("Hello Gopher!")
11
12    // write data to a hello file, with 0777 file permission
13    err := ioutil.WriteFile("hello.txt", data, 0777)
14
15    if err != nil {
16        log.Fatalf("%v", err)
17    }
18}

This will create a new file hello.txt in the current directory.

File mode

Most file systems have methods to assign permissions or access rights to specific users and groups of users. These permissions control the ability of the users to view, change, navigate, and execute the contents of the file system. – wikipedia

A few examples of filemode number notation.

0000     no permissions
0700     read, write, & execute only for owner
0770     read, write, & execute for owner and group
0777     read, write, & execute for owner, group and others
0111     execute
0222     write
0333     write & execute
0444     read
0555     read & execute
0666     read & write
0740     owner can read, write, & execute; group can only read; others have no permissions

Read a file using ReadFile

The ReadFile function reads the file by its filename and returns the file data in array of byte.

1func ReadFile(filename string) ([]byte, error)

We will read the above created hello.txt file. Please create a file if it is not created.

 1package main
 2
 3import (
 4    "fmt"
 5    "io/ioutil"
 6    "log"
 7)

 9func main() {
10
11    // read the hello.txt
12    content, err := ioutil.ReadFile("hello.txt")
13
14    if err != nil {
15        log.Fatalf("error while reading %v", err)

18    // convert the byte into string
19    fmt.Println(string(content))
20}

Output

Hello Gopher!

Write to an existing file

1. using ioutil

First using ReadFile, we will read the content and then we will append the new content.

 1package main
 2
 3import (
 4    "io/ioutil"
 5    "log"
 6)
 7
 8func main() {
 9    // Read the content
10    content, err := ioutil.ReadFile("hello.txt")
11
12    if err != nil {
13        log.Fatalf("error while reading the file. %v", err)
14    }
15    
16    //  new content
17    newText := []byte("\nAppended text.")
18    
19    // append the content
20    content = append(content, newText...)
21
22    // overwrite the content of hello.txt
23    err = ioutil.WriteFile("hello.txt", content, 0777)
24
25    if err != nil {
26        log.Fatalf("error while writing the file. %v", err)
27    }
28}

The 3 dots after newText... is a variadic argument. In simple terms it will extract all the array elements. Learn more

For Example:

1if a = [1,2,3,4]
2
3func (a...) => func (1, 2, 3, 4)

2. using os package

Using OpenFile in the os package we can open the file and append the data in it.

Package os provides a platform-independent interface to operating system functionality.

1func OpenFile(name string, flag int, perm FileMode) (*File, error)

The flag parameter defines the operation on the file. The available flags are:

    O_RDONLY int = syscall.O_RDONLY // open the file read-only.
    O_WRONLY int = syscall.O_WRONLY // open the file write-only.
    O_RDWR   int = syscall.O_RDWR   // open the file read-write.
    // The remaining values may be or'ed in to control behavior.
    O_APPEND int = syscall.O_APPEND // append data to the file when writing.
    O_CREATE int = syscall.O_CREAT  // create a new file if none exists.
    O_EXCL   int = syscall.O_EXCL   // used with O_CREATE, file must not exist.
    O_SYNC   int = syscall.O_SYNC   // open for synchronous I/O.
    O_TRUNC  int = syscall.O_TRUNC  // truncate regular writable file when opened.

Example

 1package main
 2
 3import (
 4    "log"
 5    "os"
 6)
 7
 8func main() {
 9    // If the file doesn't exist, create it, or append to the file
10    file, err := os.OpenFile("hello.txt", os.O_APPEND|os.O_CREATE|os.O_WRONLY, 0777)
11
12    if err != nil {
13        log.Fatalf("error while opening the file. %v", err)
14    }
15
16    // close the file once program execution complete
17    defer file.Close()
18
19    if _, err := file.Write([]byte("\nappended data")); err != nil {
20        log.Fatalf("error while writing the file. %v", err)
21    }
22}
23

Read and Write a JSON file

Reading and writing the JSON file requires to unmarshal and marshal the data respectively.

Learn more about How to use JSON in Golang

We are going to create a library object and a book object. The library stores the array of books.

1type book struct {
2    Name   string
3    Author string
4}
5
6type library struct {
7    Books []book
8}

Create books and library

1    // create some books
2    books := []book{
3        {Name: "An Introduction to programming in Go", Author: "Caleb Doxsey"},
4        {Name: "Go in Action", Author: "William Kennedy"},
5        {Name: "The way to Go", Author: "IVO BALBAERT"},
6    }
7
8    // create the library with books
9    lib := library{Books: books}

Write JSON

The library is in golang struct format. To convert it JSON we need to marshal it.

1    // marshal to json or convert to json
2    libJSON, _ := json.Marshal(lib)

Save it in library.json

1    // write to library.json
2    err := ioutil.WriteFile("library.json", libJSON, 0777)
3
4    if err != nil {
5        log.Fatalf("Error while writing a file %v", err)
6    }

Read JSON

Read the library.json.

1    // Read the file
2    content, err := ioutil.ReadFile("library.json")
3
4    if err != nil {
5        log.Fatalf("Error while reading a file %v", err)
6    }

The data is in byte. Unmarshal it to access the data.

 1    // create a new library
 2    var readLib library
 3
 4    // unmarshal the json to library
 5    err = json.Unmarshal(content, &readLib)
 6
 7    if err != nil {
 8        log.Fatalf("Error while unmarshal the content  %v", err)
 9    }
10
11    // print all the books
12    for i, book := range readLib.Books {
13
14        fmt.Printf("%v| Name: %-40s | Author: %s\n", i, book.Name, book.Author)
15    }

Output

0| Name: An Introduction to programming in Go     | Author: Caleb Doxsey
1| Name: Go in Action                             | Author: William Kennedy
2| Name: The way to Go                            | Author: IVO BALBAERT

The output is formatted because we used a little string formatting.
In the Printf, we have used %-40s. 40 is the width. - align the data to left. Learn more about string formatting.

Width is measured in units of Unicode code points, that is, runes.
For most values, width is the minimum number of runes to output, padding the formatted form with spaces if necessary.

Update the existing JSON file

We will use ioutil for this. First, we will read library.json and then we add a new book in it.

 1package main
 2
 3import (
 4    "encoding/json"
 5    "fmt"
 6    "io/ioutil"
 7    "log"
 8)
 9
10type book struct {
11    Name   string
12    Author string
13}
14
15type library struct {
16    Books []book
17}
18
19func main() {
20
21    // Read the file
22    content, err := ioutil.ReadFile("library.json")
23
24    if err != nil {
25        log.Fatalf("Error while reading a file %v", err)
26    }
27
28    // create a new library
29    var readLib library
30
31    // unmarshal the json to library
32    err = json.Unmarshal(content, &readLib)
33
34    if err != nil {
35        log.Fatalf("Error while unmarshal the content  %v", err)
36    }
37
38    // before update
39    fmt.Println("Before Update")
40
41    // print all the books
42    for i, book := range readLib.Books {
43
44        fmt.Printf("%v| Name: %-40s | Author: %s\n", i, book.Name, book.Author)
45    }
46
47    // Update
48    newBook := book{
49        Name:   "Go Web Programming",
50        Author: "Sau Sheong Chang",
51    }
52
53    // append the new book
54    readLib.Books = append(readLib.Books, newBook)
55
56    // write the book
57    // marshal to json or convert to json
58    libJSON, err := json.Marshal(readLib)
59
60    if err != nil {
61        log.Fatalf("Error while marshal. %v", err)
62    }
63
64    // write to library.json
65    err = ioutil.WriteFile("library.json", libJSON, 0777)
66
67    if err != nil {
68        log.Fatalf("Error while writing a file %v", err)
69    }
70
71    // After Update
72    fmt.Println("After update")
73
74    // Read the file
75    content, err = ioutil.ReadFile("library.json")
76
77    if err != nil {
78        log.Fatalf("Error while reading a file %v", err)
79    }
80
81    // unmarshal the json to library
82    err = json.Unmarshal(content, &readLib)
83
84    if err != nil {
85        log.Fatalf("Error while unmarshal the content  %v", err)
86    }
87
88    // print all the books
89    for i, book := range readLib.Books {
90        fmt.Printf("%v| Name: %-40s | Author: %s\n", i, book.Name, book.Author)
91    }
92}
